# Shivang Gupta's 3D Portfolio

A highly interactive and modern 3D portfolio project built with React, TypeScript, Three.js, and GSAP.

## Live Link
[https://shivang-portfolio-pink.vercel.app/](https://shivang-portfolio-pink.vercel.app/)

## Overview
This portfolio is personalized for **Shivang Gupta**, a Full Stack Engineer. It features:
- **Interactive 3D Scene**: Powered by React Three Fiber and Three.js.
- **Dynamic Animations**: Seamless transitions and micro-interactions using GSAP.
- **Personalized Sections**:
  - **About**: Professional overview.
  - **What I Do**: Core technical skills and domains.
  - **Experience**: Career history and educational background.
  - **Resume**: Direct preview and download of the latest CV.
- **Responsive Design**: Optimized for all devices and screen sizes.

## Local Development
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Run the development server: `npm run dev`.
4. Open [http://localhost:5173/](http://localhost:5173/) in your browser.

## Tech Stack
- React 18
- TypeScript
- Vite
- Three.js & React Three Fiber
- GSAP
- CSS Modules
